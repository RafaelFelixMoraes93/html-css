Formulários "inputs e botões"

FORM

FORM define um formulário

INPUT campo de entrada "texto, email, senha, etc"

TYPE define o tipo do input "text, email, senha, etc"

TEXTAREA campo de texto multilinha

BUTTON cria um botão "geralmente para envio"

LABEL associa um rótulo descritivo a um campo de formulário

SELECT cria um menu suspenso "dropdown"

OPTION define uma opção dentro do SELECT

NAME nome do campo "importante para envio dos dados"

PLACEHOLDER texto de dica dentro do campo INPUT
