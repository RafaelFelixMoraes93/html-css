Links e Imagens

A

Cria um link clicável para outra página, arquivo ou seção interna

HREF é definido dentro de A para destino do link

TARGET é definido de forma opcional, diz se o link será aberto em uma nova aba ou na mesma

Google

IMG

Exibe uma imagem

SRC define o caminho da imagem

ALT mostra um texto caso a imagem não carregue

WIDTH/HEIGHT define largura e altura da imagem "opcional"
