Utilizando CSS para estilização

Aqui temos algumas formas de estilização, diretamente no arquivo "não recomendado" e criando um arquivo separado que podemos reutilizar, sendo assim é a melhor forma, pois podemos reutilizar o código,

Certo, mas como faço caso tenha que utilizar dois ou mais H1 por exemplo, todos ficaram com o mesmo estilo?

Nesse caso utilizamos o "ID" ou "CLASS", servem como um identificador.

ID

Deve ser únixo por página, recomenda-se utilizar para elementsos únicos, como um botão específico ou uma seção do site.

Exemplo: #destaque {color: red}.

CLASS

Pode ser utilizado em vários elementos, recomenda-se para grupos de elementos, todos os botões, caixas de aviso, etc.

Exemplo: .importante {background-color: yellow}.

Seleções

Unindo seleções

Basta colocar vírgula.

Exemplo: #destaque, .importante {color: blue}.

Colocando estilos diferentes em seleções unidas

Exemplo: #destaque {font-weight: normal}.

Exemplo: .importante {font-weight: boldo}.

Acessando elementos dentro de elementos

Exemplo: ol li {}

Nesse caso, ele irá até todas as "ol" não não as modificará, e irá apenas mudar os os itens "li" que estão dentro daquelas listas.
