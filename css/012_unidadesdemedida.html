Unidades de medidas

As unidades de medidas são utilizadas para determinar o tamanho das coisas com css.

Medidas Absolutas

Não sofrem alterações alterações de outros elementos, são absolutas, então se forem colocados 450px será um valor exato.

Temos o "pt" 450pt, são um pouco maior que o px, porém a mais utilizada é o px.

Temos o "in" esse tem seu tamanho em polegadas.

Temos "cm" e "mm" que são unidades em centrimetros e milimetros.

Vale resaltar que os pixels do CSS são diferentes dos pixels dos monitores, são um pouco maior em relatividade aos monitores.
