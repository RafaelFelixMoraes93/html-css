Herança no CSS

Em CSS, a herança é o mecanismo de passar propriedadas de estilo de forma automática do elemento pai para os elementos filhos, menos quando são sobrescritos.

Nem todas as propriedades CSS são herdadas, as mais comuns são:

  • color
  • font-family
  • font-size
  • line-height
  • text-align
  • visibility

As que não são herdadas

  • background-color
  • border
  • margin
  • padding

Exemplo

CSS

.pai {

color: blue;

font-family: Arial.

}

HTML

>div class="pai">
